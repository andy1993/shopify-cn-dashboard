FAQ

常见问题

关于 Shopify CN Pro 你想知道的一切

这是完全免费的吗？

是的，Shopify CN Pro 以 MIT 协议完全开源，永久免费商用。你可以自由 fork、部署、修改，无任何订阅费用或隐藏收费。

我没有 Shopify 店铺，能体验吗？

完全可以。系统内置「一键导入演示数据」功能，包含 2 家高保真虚拟店铺、14 天历史订单、50+ 模拟客户，无需 Shopify Token 即可完整体验全部 47 个面板。

我的店铺 Token 安全吗？

你的 Admin API Token 仅保存在浏览器本地 LocalStorage，不会上传至任何第三方服务器。系统通过 Next.js API Route 在服务端安全代理所有 Shopify 请求，后端无数据库、无持久化存储。

AI 智能诊断需要额外付费吗？

AI 诊断功能需要你提供自己的 DeepSeek API Key（费用极低，通常每次诊断不超过几分钱）。若没有 Key，系统会自动优雅降级到高保真本地预设报告，不影响其他所有功能正常使用。

支持多店铺管理吗？

支持。你可以在配置页添加多个 Shopify 店铺，「全店聚合大盘」面板会用 Promise.all 并发拉取所有店铺数据并做横向对比分析。

如何部署到生产环境？

执行 npm run build && npm start 即可。系统基于 Next.js 16，可一键部署到 Vercel、Railway、自有 VPS 等任意 Node.js 环境，无需数据库或额外中间件。

支持哪些 Shopify API 版本？

当前基于 Shopify 2026-04 Stable API，同时覆盖 REST Admin API（订单/库存/履约）与 GraphQL Admin API（商品/Markets/变体改价）双协议，确保与最新 Shopify 后台保持同步。

我可以在开源协议范围内商业化使用吗？

可以。MIT License 允许你将本项目用于任何商业用途，包括为客户部署、二次开发销售等，唯一要求是保留原始 MIT 版权声明。
